、# Linkding Chrome 插件 3.0 升级说明

> 版本：3.0 | 更新日期：2026-03-26

---

## 一、如何从 2.0 升级到 3.0

1. 打开 Chrome，进入 `chrome://extensions`
2. 找到已安装的 2.0 插件，点击**移除**
3. 点击右上角**加载已解压的扩展程序**
4. 选择 `linkding_chrome3.0` 文件夹
5. 安装完成后，点击插件图标进入**设置**，重新填写服务器地址和 API Token 并保存

> ⚠️ 升级后需重新配置设置，2.0 的设置不会自动迁移。

## 二、Bug 修复

| 问题 | 2.0 状态 | 3.0 修复 |
|------|---------|---------|
| `manifest.json` 包含非法 JS 注释 | ❌ 导致 Chrome 解析报错 | ✅ 已移除所有注释 |
| `popup.html` 底部有残留重复代码块 | ❌ `</html>` 后多余 `<div>` | ✅ 已清理 |
| `style.css` 大量重复样式定义 | ❌ `.form-group`、`label`、`input` 各定义两遍 | ✅ 去重合并 |
| 打开 `chrome://` 系统页面时插件报错 | ❌ 抓取描述时报 `cannot access chrome://` | ✅ 已跳过系统页面 |
| 重复书签检测性能差 | ❌ 拉取全部书签后手动过滤 | ✅ 改用官方 `/api/bookmarks/check/` 接口 |
| 提交后按钮无反馈 | ❌ 点击后无任何提示 | ✅ 按钮变为"保存中…"并禁用，防止重复提交 |
| `manifest.json` 包含非法权限 `"action"` | ❌ 权限列表中有无效项 | ✅ 已移除 |

## 三、新增功能

### 3.1 快捷标签按钮

标签输入框下方新增一排快捷标签按钮，方便一键打标签：

- **新书签**：显示在设置里配置的「固定快捷标签」，默认未选中，点击即选中
- **旧书签（已收藏过的 URL）**：显示该书签已有的标签，默认全部选中
- 点击已选中的按钮可取消，标签输入框实时同步
- 设置项：**快捷标签数量**（1~10，默认 3）、**固定快捷标签**（空格隔开）

### 3.2 「+ 更多标签」下拉框

快捷标签按钮旁边新增一个下拉选择框，列出所有标签供选择：

- 下拉框里的标签按以下规则排序：
  - **字母/数字开头的标签排在最前面**（A → Z）
  - **中文标签排在后面**（按系统排序）
- 选中后自动加入标签输入框，下拉框重置回"+ 更多标签"

**⚠️ 注意：如果下拉框里出现了网站上没有的标签关键词，请检查 Linkding 后台，删除书签数为 0 的标签。**

> 操作路径：进入 Linkding 网站 → 左侧标签栏 → 找到书签数为 0 的标签 → 删除

### 3.3 推荐标签（可在设置中开关）

在快捷标签下方独立显示一行推荐标签（橙黄色虚线按钮，与快捷标签视觉区分）：

推荐逻辑分两部分：

**① 固定推荐**：在设置里填写的「固定推荐标签」，每次打开插件都会显示（已选中的除外）

**② 智能推荐**：根据当前页面的**标题**和**描述**，与你所有的标签做匹配，自动推荐相关标签。匹配方式：
- 标签名整体出现在标题/描述中（如描述含"Chrome"，推荐标签"Chrome"）
- 标题/描述里的关键词包含于标签名（如标题含"书签"，推荐标签"书签类开发"）
- 标签名的子词出现在标题/描述中（如标签"浏览器插件"，描述含"浏览器"）

点击推荐标签后，会自动加入输入框并消失。

## 四、设置页新增配置项

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 快捷标签数量 | 快捷按钮最多显示几个（1~10） | 3 |
| 固定快捷标签 | 每次新书签时显示的固定标签（空格隔开） | 空 |
| 开启推荐标签 | 是否显示推荐标签区域 | 关闭 |
| 固定推荐标签 | 每次都推荐的标签（空格隔开），如"未分类 办公效率" | 空 |

## 六、关于「更多标签」出现陌生标签的排查方法

如果你发现「+ 更多标签」下拉框里出现了一些不认识的标签，或者已经删除的标签还在出现，原因通常是：

> **Linkding 里存在书签数为 0 的"空标签"**——标签曾经被用过，但关联的书签都删了，标签本身没有跟着清理。

**清理方法：**

1. 登录 Linkding 网站
2. 点击左侧标签列表，找到书签数量显示为 **0** 的标签
3. 点击该标签名旁边的删除按钮，将其删除
4. 删除后，重新打开插件，下拉框里的陌生标签就消失了

> 💡 建议定期清理 0 书签标签，保持标签库干净，推荐效果也会更准确。
