配置

这是 Linkding 书签服务的配套扩展程序。在开始使用之前，您需要配置一些基本设置，以便扩展程序能够与您的 Linkding 安装进行通信。

填写基本信息：

Linkding 安装的基本 URL，不包含 /bookmark 路径或尾部斜杠。
用于对 Linkding API 进行身份验证。您可以在 Linkding 设置页面找到此令牌。
默认添加到新书签的标签。（当开启新旧书签对比时此功能失效）

添加书签时预选：

默认将新书签标记为未读。

保存书签时预选共享：

默认将新书签标记为共享。仅当您在 Linkding 中启用了书签共享功能时才有用。

检查重复书签：

添加新书签时，自动检查是否存在相同网址的书签，若存在则同步旧书签信息。
